# 一键安装管理脚本

 1.功能：
 包含SSR服务端后端+宝塔面板+BBRPlus一键安装
 服务管理
 仅支持Centos 7

 2.安装命令(CentOS)
 wget -N https://raw.githubusercontent.com/AntonyWilsonClub/SSR/master/MENU.sh && chmod +x MENU.sh
 
 3.运行命令
 ./MENU.sh
